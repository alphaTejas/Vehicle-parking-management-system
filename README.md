   ```bash
   python3 -m venv .venv
   ```

3. **Activate the virtual environment**
   - **Mac/Linux:**
     ```bash
     source .venv/bin/activate
     ```
   - **Windows (Command Prompt):**
     ```bash
     .venv\Scripts\activate
     ```

     ```bash
     .\.venv\Scripts\Activate.ps1
     ```

   ```bash
   pip install -r requirements.txt
   ```

   ```bash
   python app.py
   ```
